### R 언어의 특징

    1.데이터 분석에 특화되어 있다.

    2.탄탄한 사용자 커뮤니티를 보유하고 있다.

    3.다양한 패키지(라이브러리)를 제공한다.

    4.미적이고 기능적인 통계 그래프를 제공한다.

    5.편리한 프로그래밍 환경

### R 설치
본인은 맥북을 사용중이므로 패키지 매니저인 Homebrew를 사용했다.
```Terminal
~brew install R
```
### R 스튜디오 설치
